## 🚀 Projects

### 🌌 The Primordia Bot Ecosystem
> A trio of interconnected Discord bots built around an original fantasy universe called **Primordia** — complete with custom lore, characters, realms, and events.

| Bot | Role | Features |
|-----|------|----------|
| [**great-one**](https://github.com/MurtazaJafri-27/great-one) | The Core | Daily lore quests, XP & leveling system (20 levels), character/lore lookup, customizable player profiles |
| [**keen-keeper-**](https://github.com/MurtazaJafri-27/keen-keeper-) | The Archivist | Moderation logging, message deletion tracking, kick/ban logs, media-based leveling with role rewards |
| [**gremlin-bot**](https://github.com/MurtazaJafri-27/gremlin-bot) | The Watcher | Welcome system, button-based verification, lore broadcast embeds |

All three bots are built with **Discord.js** and themed around the original Primordia lore — even the error messages are in-universe.
